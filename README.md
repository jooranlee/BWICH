# BWICH

This is the repo for the manuscript "B-WICH Complex Cooperates with the RNA Exosome to Resolve Transcriptional Onco-Condensates".

- **clusterProfiler** (v4.12.6): A universal enrichment tool for interpreting omics data
- **circlize** (v0.4.16): Circular Visualization
- **ComplexHeatmap** (v2.20.0): Make Complex Heatmaps
- **data.table** (v1.17.8): Extension of `data.frame`
- **DESeq2** (v1.44.0): Differential gene expression analysis based on the negative
binomial distribution
- **dplyr** (v1.1.4): A Grammar of Data Manipulation
- **fgsea** (v1.30.0): Fast Gene Set Enrichment Analysis
- **ggplot2** (v4.0.0): Create Elegant Data Visualisations Using the Grammar of Graphics
- **ggpattern** (v1.2.1): 'ggplot2' Pattern Geoms
- **ggpubr** (v0.6.2): 'ggplot2' Based Publication Ready Plots
- **ggrepel** (v0.9.6): Automatically Position Non-Overlapping Text Labels with
'ggplot2'
- **org.Hs.eg.db** (v3.19.1): Genome wide annotation for Human
- **qusage** (v2.38.0): qusage: Quantitative Set Analysis for Gene Expression
- **readr** (v2.1.5): Read Rectangular Text Data
- **readxl** (v1.4.5): Read Excel Files
- **reshape** (v0.8.10): Flexibly Reshape Data

- **SingleCellExperiment** (v1.26.0): S4 Classes for Single Cell Data

- **TCGAbiolinks** (v2.32.0): TCGAbiolinks: An R/Bioconductor package for integrative analysis
